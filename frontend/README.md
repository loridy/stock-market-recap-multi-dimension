# Frontend Display (Canonical Path)

The canonical frontend is now:
- `index.html` at repository root (`/` when served)

Legacy path compatibility:
- `frontend/index.html` only redirects to `/`.

Frontend reads canonical JSON output from pipeline:
- `reports/YYYY-MM-DD/report.json`

Recommended sections:
- Executive Summary
- Regime Dashboard
- Sector & Factor views
- Analyst focus cards
- Risks and Next Actions
