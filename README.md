# Stock Market Recap (Multi-dimension)

A project to build a multi-dimensional stock market recap framework that summarizes cross-asset moves, drivers, and actionable investment insights.

## Initial Structure
- `data/` raw/intermediate datasets
- `notebooks/` exploratory analysis notebooks
- `reports/` output summaries and recap artifacts
- `scripts/` reusable data/analysis scripts
- `docs/` project notes and methodology
